Проект, состоящий из двух сцен:
1 сцена:
  - в центре кнопка Next, по клику на которую, открывается вторая сцена
2 сцена:
  - слева кнопка, по клику на которую, изменяется её цвет
  - по центру кнопка со счетчиком, по клику изменяется значение счетчика
  - справа список из 10 различных картинок, который можно скроллить
  - внизу слева кнопка Back, по клику на которую происходит возврат на первую сцену
